# HR-Workforce-Diversity-Attrition-Dashboard--Tableau-Project

Interactive Tableau dashboard analyzing workforce diversity, employee demographics, attrition trends, tenure patterns, and department-wise turnover using HR analytics and data visualization.

Overview

This Tableau project analyzes workforce diversity, employee demographics, and attrition patterns to help HR teams understand retention risks and workforce composition.

The dashboard provides insights into employee distribution, gender balance, tenure trends, department-wise attrition, and exit reasons through interactive visualizations.

---

Objectives

- Analyze workforce diversity across departments
- Monitor employee attrition trends
- Identify high-risk attrition segments
- Understand employee demographics and tenure distribution
- Build an interactive HR analytics dashboard using Tableau

---

Tools & Technologies

- Tableau
- Microsoft Excel
- Data Visualization
- HR Analytics

---

Dashboard Features

- KPI Cards (Total Employees, Attrition Rate, Avg Tenure, Gender Ratio)
- Gender Distribution Pie Chart
- Employee Age Distribution
- Department-wise Gender Split
- Attrition Trend Analysis
- Attrition Heatmap
- Exit Reason Analysis
- Interactive Filters & Drill-downs

---

Key Insights

- Identified departments with higher attrition rates
- Analyzed workforce diversity distribution
- Evaluated employee tenure patterns
- Compared attrition trends across employee groups
